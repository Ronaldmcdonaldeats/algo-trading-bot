================================================================================
RUNNING THE BOT:
================================================================================

Basic run (headless):
  docker run --rm -it --env-file .env -v ./data:/app/data algo-trading-bot:latest \
    python -m trading_bot paper run --symbols AAPL,MSFT --period 6mo --no-ui

With multiple iterations:
  docker run --rm -it --env-file .env -v ./data:/app/data algo-trading-bot:latest \
    python -m trading_bot paper run --symbols AAPL,MSFT,GOOGL --iterations 10 --no-ui

Specific period and interval:
  docker run --rm -it --env-file .env -v ./data:/app/data algo-trading-bot:latest \
    python -m trading_bot paper run --symbols AAPL --period 1y --interval 1d --no-ui

Backtest mode:
  docker run --rm -it --env-file .env -v ./data:/app/data algo-trading-bot:latest \
    python -m trading_bot backtest run --symbols AAPL,MSFT --period 1y
