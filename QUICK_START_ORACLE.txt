# QUICK START CARD - ORACLE CLOUD BOT SETUP

## WHAT YOU NEED
- [ ] Oracle Cloud instance IP address
- [ ] SSH key file (.key)
- [ ] Alpaca API Key (from https://app.alpaca.markets/account/settings)
- [ ] Alpaca Secret Key (same page)

---

## 5 COMMANDS TO RUN

### Command 1: SSH into instance
```bash
ssh -i C:\path\to\key.key ubuntu@YOUR_IP
```
**Replace:** `C:\path\to\key.key` with your key location
**Replace:** `YOUR_IP` with your Oracle instance IP (example: 129.213.45.123)

---

### Command 2: Download setup script
```bash
cd /home/ubuntu && curl -O https://raw.githubusercontent.com/YOUR_USERNAME/algo-trading-bot/main/deploy_oracle.sh && bash deploy_oracle.sh
```

⚠ **If GitHub doesn't work**, I'll provide the script content below.

---

### Command 3: Add your Alpaca credentials
```bash
nano /home/ubuntu/trading-bot-cloud/.env
```

Find these lines and replace with YOUR actual keys:
```
APCA_API_KEY_ID=YOUR_ALPACA_API_KEY_HERE
APCA_API_SECRET_KEY=YOUR_ALPACA_SECRET_KEY_HERE
```

Save: `Ctrl+X` → `Y` → `Enter`

---

### Command 4: Copy bot code
```bash
cd /home/ubuntu/trading-bot-cloud
git clone https://github.com/YOUR_USERNAME/algo-trading-bot.git .
```

Or copy manually from your computer:
```bash
# FROM YOUR COMPUTER (new terminal):
scp -i C:\path\to\key.key -r C:\Users\Ronald\projects\algo-trading-bot/* ubuntu@YOUR_IP:/home/ubuntu/trading-bot-cloud/
```

---

### Command 5: Build & start bot
```bash
cd /home/ubuntu/trading-bot-cloud
docker build -t trading-bot-cloud:latest .
docker-compose up -d
```

✅ **DONE! Bot running 24/7!**

---

## VERIFY IT'S WORKING

```bash
docker logs -f trading-bot-cloud
```

You should see:
- `[ITERATION X]` messages
- `[SIGNAL]` messages with stock symbols
- `[PORTFOLIO]` showing your positions

Press `Ctrl+C` to exit (bot keeps running)
